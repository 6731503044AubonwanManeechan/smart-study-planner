# 📚 Smart Study Planner

**Smart Study Planner** is a productivity mobile application built with Flutter.
It helps students organize tasks, manage study schedules, and track their progress efficiently.

---

## 🎯 Objective

This application is designed to:

* Help students plan their study schedule effectively
* Reduce missed deadlines and forgotten tasks
* Improve time management and productivity

---

## 🚀 Features

### 📝 Task Management

* Add, edit, and delete tasks
* Set start date and end date/time
* Mark tasks as completed ✔️

### 📅 Calendar View

* View all tasks in a calendar format
* Easily track deadlines and schedules

### 🔔 Notifications

* Get reminders before deadlines
* Toggle notifications on/off

### 🌐 Multi-language Support

* Supports multiple languages (e.g., Thai / English)

### 🎨 User Interface

* Clean and user-friendly design
* Supports Light and Dark mode

---

## 🛠 Tech Stack

* **Flutter** — Frontend framework
* **Dart** — Programming language
* **Firebase Firestore** — Database
* **Firebase Authentication** — User authentication
* **Git & GitHub** — Version control
